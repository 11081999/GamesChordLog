Requerimientos Funcionales


  • Recomendacón de música con base en gustos de videojuegos.
  • Conección con Steam API y Spotify API.
  • Manejo de cuentas de ususario.
  • Autorizacion y niveles de acceso.
  • Inicio de sesión con facebook y google.
  • Cuentas de administradores pueden modificar otras cuentas menos las de superadmin.
  • Las cuentas de super-admin pueden modificar todas las cuentas.
  • Las cuetas de admin y super-admin pueden borrar cuentas.
  • Solo las cuentas de super-admin pueden borrar cuentas de admins.
  • Las cuentas de ususarios NO pueden modificar otras ceuntas, solo la suya.
  • Se debe de guardar en una persistencia los datos coneguidos desde las APIs del usuario loggeado.

Requerimientos NO Funcionales


  • Interfaz atractiva con temática de videojuegos.
  • la interfaz se adapta dependiendo del nivel de acceso del usuario.
  • Funciona correctamente y sin errores o bugs críticos.
  • Tiempo de carga rápido y minima espera.
  • Métricas de rendimiento con google analytics.
